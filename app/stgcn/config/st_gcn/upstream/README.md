# Upstream-конфиги ST-GCN

Файлы перенесены сюда, чтобы в корне `config/st_gcn/` оставались только **активные** конфиги проекта (`ntu-xsub-kinetics-2d`, `kinetics-skeleton`).

При запуске из `app/stgcn` указывайте полный путь, например:

```bash
python main.py recognition -c config/st_gcn/upstream/ntu-xsub-2d/train.yaml
```
