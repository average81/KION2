# Данные для ST-GCN (локально)

Эти каталоги нужны, чтобы **обучать и проверять** ST-GCN на **2D**-скелетных последовательностях (задача распознавания действий по позам из NTU RGB+D и т.п.). **Сборка датасета в формат для обучения** (скрипты, пути, сплиты под `ntu-xsub-kinetics-2d`) — разработка проекта KION2; см. корневой `README.md`. Сырые скелеты и собранные массивы не коммитятся из‑за размера.

В репозитории **не хранятся** большие файлы: `*.skeleton`, `*.npy` в этой ветке игнорируются через корневой `.gitignore`.

Ожидаемая структура (после распаковки NTU и сборки скриптом `tools/build_ntu_xsub_2d_25_18.py`):

| Путь | Содержимое |
|------|------------|
| `NTU-RGB-D/nturgb+d_skeletons/*.skeleton` | исходные 2D color скелеты NTU RGB+D |
| `NTU-RGB-D-2D-from-color/xsub_*_data18.npy`, `*_data25.npy`, `*_label.pkl` | выход сборки для обучения (пути в `config/st_gcn/ntu-xsub-kinetics-2d/*.yaml`) |

Корень данных в скрипте сборки: `models/stgcn/data/` (см. `tools/build_ntu_xsub_2d_25_18.py`).

Чекпоинты и логи обучения пишутся в `models/stgcn/work_dir/` — тоже в `.gitignore`.
